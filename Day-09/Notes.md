What is a List?

A list is an ordered, mutable collection of elements.

Creating a List
numbers = [10, 20, 30, 40]

Accessing Elements
numbers[0]   # first element
numbers[-1]  # last element

Modifying List
numbers[1] = 25

Common List Methods

append() – add item

insert() – add at position

remove() – delete item

pop() – remove by index

len() – list length
