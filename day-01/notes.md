## What is Python?
Python is a high-level, interpreted programming language that is easy to read,
easy to write, and beginner-friendly.

It was created by **Guido van Rossum** and is widely used in industry today.

Python is used in:
- Web Development

- Artificial Intelligence & Machine Learning

- High demand in jobs
